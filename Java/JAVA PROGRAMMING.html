JAVA PROGRAMMING

CMDBID: 736

Course Code: JAVA101

Duration: 3 Days

COURSE OVERVIEW

This Java Programming course is designed to equip participants with the skills and knowledge needed to support, maintain, and debug existing Java applications while also enabling them to make enhancements to source raw performance data from various sources such as data files, APIs, and SQL queries, and update it into a PostgreSQL database.

Overview

The course provides comprehensive coverage of Java programming concepts, focusing on practical applications and real-world scenarios. Participants will learn essential skills to become proficient Java developers capable of handling complex projects.

Audience

This course is intended for individuals with intermediate-level knowledge in Java programming. It is suitable for:

  • Java developers
  • Software engineers
  • Technical professionals involved in Java application development

Skills Gained

Upon completion of this course, participants will gain the following skills:

  • Supporting and maintaining existing Java applications
  • Debugging Java code effectively
  • Enhancing Java applications to source raw performance data from various sources
  • Updating performance data into a PostgreSQL database

Prerequisites

Participants should have intermediate-level knowledge in Java programming. Prior experience in supporting, maintaining, and debugging Java applications is beneficial.

Outline

The course covers the following topics:

  • Eclipse
  • Language Fundamentals
  • Objects and Classes
  • Using Java Objects
  • Inheritance in Java
  • Advanced Inheritance and Generics
  • Packages
  • Exception Handling
  • Input/Output Streams
  • Core Collection Classes
  • Collection Sorting and Tuning
  • Inner Classes
  • Introduction to Swing
  • Swing Events and Layout Managers
  • Introduction to JDBC
  • JDBC SQL Programming
